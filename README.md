# Use of AI in Hyperactive Social Media Advertisements
### Effects on Brand Perception and Consumer Response

**IAPC Project — Semester VI | Cluster Innovation Centre, University of Delhi**  
Supervised by Dr. Sanjeewani Sehgal

---

## About

This study investigates how AI-driven, algorithmically personalised social media advertising affects consumer psychology and behaviour. Using the **S-O-R (Stimulus-Organism-Response)** framework, we examine two key outcomes:

- **Ad-Avoidance Behaviour** — when hyper-targeted ads feel like surveillance
- **Performative Sharing** — when consumers share ads as social/identity currency

Key constructs explored: `Algorithmic Paranoia` · `Algorithmic Serendipity` · `Meme-jacking` · `Information Fatigue` · `Message Ephemerality`

---

## Key Findings

| Hypothesis | Path | β | Result |
|---|---|---|---|
| H1a | Paranoia → Ad-Avoidance | 0.421*** | ✅ Supported |
| H1a | Info Fatigue → Ad-Avoidance | 0.369*** | ✅ Supported |
| H1b | Serendipity → Ad-Avoidance | -0.117 | ❌ Rejected |
| H1c | Paranoia × Serendipity (moderation) | 0.034 | ❌ Rejected |
| H2a/H2c | Meme-jacking → Performative Sharing | 0.500*** | ✅ Supported |
| H2b | Message Ephemerality → Sharing | 0.337*** | ✅ Supported |

- Model 1 (Ad-Avoidance): **R² = 0.434**
- Model 2 (Performative Sharing): **R² = 0.545**

---

## Methodology

- **Design:** Quantitative, cross-sectional survey (N = 129)
- **Tool:** Structured questionnaire via Google Forms (7-section, 5-point Likert scale)
- **Analysis:** PLS-SEM with OLS regression, HC3 robust standard errors, moderation via mean-centred interaction terms
